Getting Started #back to top

When you purchase Colova from Themeforest, you need to login your account and download theme files from Themeforest. Please make sure you downloaded All file & Documentation option.

What is included?

After you downloaded theme files you will find inside it;

Documentation - Contains Colova template documentation

colova.zip - Template file for installation.

Template Installatıon #back to top

Step 1: Login to your hosting server via FTP client.
Step 2: Locate the domain folder you wish to upload Colova.
Step 3: Unzip the file "colova.zip".
Step 4: Upload the unzipped files in your domain server.

Header Options #back to top

Header Color Settings #back to top

There is two header color set included in Colova.
1: Dark Header (Default Color Set)
2: Light Header

For the change the header appearance to light; you need to add header-light class after site-header class.

Site Logo #back to top


Add your logo url in the section viewed below.

For a good appearance in all color sets. Add your logo url both light and dark logo sections.

Menu Settings #back to top

There is two menu styles included in Colova.
1: Classic Menu
2: Fullscreen Menu

For the change the menu appearance; you need to add a class after site-navigation class.

Add navigation-classic class for classic menu style.
Add navigation-fullscreen class for fullscreen menu style.

Scroll Effects #back to top

It's very easy to add an animation any element in Colova.
You need to add just two classes the which element you want to animate.
These are has-animation and animation name.
Usable animations;

fade-in
fade-in-up
fade-in-down
fade-in-right
fade-in-left
